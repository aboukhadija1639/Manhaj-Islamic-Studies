# وحدة علوم القرآن

## نظرة عامة

تم إضافة وحدة علوم القرآن الكريم إلى منصة منهاج للدراسات الإسلامية. هذه الوحدة توفر تجربة تعليمية حديثة وشاملة مع دعم كامل للغة العربية (RTL).

## المحتوى

تحتوي الوحدة على:
- محاضرات الدكتور مخلوف في مقياس علوم القرآن - الجزء الأول
- محاضرات علوم القرآن - الجزء الثاني

## الميزات الرئيسية

### واجهة المستخدم
- ✅ تصميم يدعم اللغة العربية بالكامل (RTL)
- ✅ واجهة سريعة الاستجابة (موبايل، تابلت، سطح المكتب)
- ✅ قائمة جانبية قابلة للطي مع أقسام الدروس
- ✅ بحث فوري مع نتائج لحظية
- ✅ جدول محتويات مع مزامنة التمرير

### البحث والاكتشاف
- ✅ بحث نصي كامل مع مطابقة تقريبية
- ✅ معالجة النص العربي وإزالة التشكيل
- ✅ ترتيب النتائج حسب الأهمية
- ✅ اختصار لوحة المفاتيح (Ctrl+K)

### تتبع التقدم
- ✅ حالة إكمال الدرس
- ✅ حساب تقدم الوحدة
- ✅ تتبع الوقت المستغرق
- ✅ حفظ محلي (localStorage)

### إمكانية الوصول
- ✅ متوافق مع معايير WCAG AA
- ✅ التنقل بلوحة المفاتيح
- ✅ تسميات ARIA
- ✅ دعم قارئات الشاشة

## البدء السريع

### 1. توليد الـ Manifest

```bash
npm run generate:manifest
```

### 2. تشغيل خادم التطوير

```bash
npm run dev
```

### 3. الوصول إلى الوحدة

افتح المتصفح على:
```
http://localhost:5173/modules/ulum-al-quran
```
