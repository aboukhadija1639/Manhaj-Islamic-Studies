# مقياس الفقه الإسلامي - Fiqh Module

## نظرة عامة

مقياس الفقه الإسلامي هو مقياس أساسي في برنامج جذع مشترك شريعة إسلامية (ليسانس سنة أولى). يهدف المقياس إلى تعريف الطلاب بأساسيات الفقه الإسلامي وتطبيقاته العملية.

## محاور المقياس

### 1. مدخل إلى الفقه الإسلامي
- تعريف الفقه لغة واصطلاحاً
- أهمية الفقه الإسلامي
- موضوعات الفقه الرئيسية

### 2. مصادر الفقه الإسلامي
- القرآن الكريم
- السنة النبوية
- الإجماع
- القياس

### 3. الفقه الإسلامي - العبادات
- الصلاة وأحكامها
- الزكاة وشروطها
- الصيام وأحكامه
- الحج وأركانه

### 4. الفقه الإسلامي - المعاملات
- البيع والشراء
- الإجارة والإيجار
- الشركة والعقود
- القرض والدين

### 5. الفقه الإسلامي - الأحوال الشخصية
- الزواج والخطبة
- الطلاق والعدة
- الميراث والتركة

### 6. الفقه الإسلامي - الجنايات والحدود
- الحدود الشرعية
- التعزيرات
- الديات والقصاص

## الميزات

### ✅ اختبار تفاعلي شامل
- 50 سؤال متنوع (اختيار من متعدد + صح/خطأ)
- تغطية شاملة لجميع محاور المقياس
- تصحيح فوري مع شرح لكل إجابة

### ✅ نظام تتبع التقدم
- حفظ نتائج الاختبارات محلياً
- إحصائيات مفصلة للأداء
- تحليل التحسن والاستقرار

### ✅ واجهة حديثة وسهلة الاستخدام
- تصميم RTL-first
- دعم كامل للغة العربية
- استجابة على جميع الأجهزة

### ✅ مراجع أكاديمية موثوقة
- الوجيز في شرح القواعد الفقهية
- الفقه الإسلامي والمعاملات
- أصول الفقه الإسلامي
